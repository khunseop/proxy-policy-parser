# UI 개선 할일 목록

> 우선순위 순으로 정렬. 완료 항목은 ✅ 표시.

---

## 1. 상세 패널 넓이 확대 + 사이드바/패널 리사이즈 `[HIGH]`

**문제**: 메인 컬럼에서 조건(Condition)·액션(Actions) 텍스트가 잘려 가시성이 낮음.  
메인 리스트에서 실질적으로 참조하는 정보는 정책명뿐이므로, 상세 패널을 더 넓게 잡아도 됨.

**할일**:
- [ ] 상세 패널 기본 너비 360px → 480px 확대
- [ ] 상세 패널 좌측 경계에 드래그 리사이저 추가 (300px ~ 700px 범위)
- [ ] 좌측 사이드바 우측 경계에도 드래그 리사이저 추가 (180px ~ 400px 범위)

---

## 2. 트리 사이드바 정책명 잘림 개선 `[HIGH]`

**문제**: 사이드바 트리에서 긴 정책명이 말줄임표(`…`)로 잘림.  
현재 `title` 속성은 있으나 툴팁이 충분하지 않음.

**할일**:
- [ ] 호버 시 말풍선(tooltip) 스타일 커스텀 툴팁 표시 (브라우저 기본 `title` 대신)
- [ ] 트리 노드 호버 시 잘린 이름 full-text를 상단/하단 플로팅 박스로 표시
- [ ] 사이드바 리사이저로 너비 조절 가능하게 (위 항목 1과 연계)

---

## 3. 메인 리스트 조건·액션 표시 개선 `[MEDIUM]`

**문제**: 현재 조건식이 70자로 잘려 표시되고 single-line이라 복잡한 조건은 알아보기 어려움.

**할일**:
- [ ] 조건식 미리보기 텍스트를 2줄까지 허용 (white-space: normal, line-clamp: 2)
- [ ] 조건이 "Always"가 아닌 경우 별도 색상 강조 (현재는 회색 처리)
- [ ] Actions도 row-meta에 간략 표시 추가 (현재는 상세 패널에서만 확인 가능)

---

## 4. 검색·전체 결과 Excel(CSV) 내보내기 `[MEDIUM]`

**문제**: 검색 결과나 전체 정책 목록을 외부로 추출할 방법이 없음.

**할일**:
- [ ] 메인 툴바에 "내보내기" 버튼 추가 (현재 표시 중인 결과 기준)
- [ ] 내보내기 항목: Name, Type, Enabled, Path, Condition, Actions
- [ ] 파일명: `policies-{setId}-{timestamp}.csv` (BOM 포함, Excel 호환)
- [ ] 전체보기·검색결과·드릴다운 결과 모두 지원

---

## 5. 검색 성능 개선 `[MEDIUM]`

**문제**: 정책 세트 선택 시 `loadObjects()`가 모든 list 엔트리를 한 번에 fetch해서 리스트가 많으면 초기 로딩이 느림.

**할일**:
- [ ] `loadObjects()` 지연 로딩으로 전환: 처음에는 list 메타(id, name, count)만 로드
- [ ] List 엔트리는 해당 List 선택 시 on-demand 로드
- [ ] 백엔드에 `/api/v1/objects/{set_id}/meta` 엔드포인트 추가 (엔트리 없이 메타만 반환)
- [ ] 전체정책 `loadAllPolicies()` 결과를 메모리에 캐시 (같은 세트 재선택 시 재요청 없음)
